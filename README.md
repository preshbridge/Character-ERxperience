# Interactive Character Showcase Game

## Description
This Unity project allows players to select different characters and play animations. Each character has an idle animation and two action animations. Users can also control music volume and toggle audio.

## Controls
- **Play Button**: Loads the GameScene
- **Options Button**: Opens options panel
- **Exit Button**: Quits game
- **Character Buttons (A/B/C): Activates a character and plays animations
- **Action Buttons (if any): Trigger character-specific actions
- **Audio Slider**: Adjusts music volume
- **Audio Play/Pause**: Toggles music

## Scenes
- IntroScene/MainMenu: Select characters, access options
- GameScene: Gameplay with characters and animations

## How to play:
1. Open the project in Unity
2. Press Play
3. Use the buttons as described above
